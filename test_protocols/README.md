# Test Clinical Trial Protocols

Example protocol documents for end-to-end testing of NIX AI analysis pipeline.

## Files

| File | Phase | Therapeutic Area | Description |
|------|-------|-----------------|-------------|
| `phase3_oncology_protocol.txt` | Phase 3 | Oncology | Non-small cell lung cancer immunotherapy trial |
| `phase2_cardiology_protocol.txt` | Phase 2 | Cardiovascular | Heart failure treatment with novel compound |
| `phase1_neurology_protocol.txt` | Phase 1 | Neurology | Early-stage Alzheimer's disease dose-finding |

## Usage

1. Upload any of these files via the NIX AI Upload dialog
2. Click **Run Regulatory Analysis** in the top bar
3. Review FDA/EMA compliance findings, jurisdiction scores, and payer gaps
4. Test the chat interface with questions about the protocol
5. Try amendment simulation and protocol comparison features
